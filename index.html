Stage Zero - Getting Started with HTML

What is the web?

The internet was created in the 1980s and now has more than 30 billion pages, mostly HTML documents. It is the largest computer network in the world. Web pages are stored on servers. Browsers on indiviual computers send requests for pages in HTTP through the internet to servers.

What is HTML?

HTML, short for hyper text markup language, is the main document type of the web, and is considered the "glue" of the web. The text content is what is being viewed, and the HTML markup dictates how that content appears on the screen.

Elements and Tags

HTML documents are made of elements. Tags denote the beginning and end of elements. However, there are many void elements which do not require an end tag. Attributes modify elements.

Elements can be inline, block, or container elements. Inline elements relate to that line of the document. Block elements form an invisible box around the text they contain, and can have associated height and width.

Computers are stupid

Ok, computers are not really stupid. They simply operate based upon a given set of rules, and require precise input to operate as expected.
